O projeto tinha como objetivo extrair caracteristicas da base de dados JAFFE (The Japanese Female Facial Expression (JAFFE) Dataset). Porém o projeto pode ser utilizado em outras bases de dados. Pelo uso da PCA, é possível classificar as imagens pela análise das características extraidas.
